# Music Generator Frontend

React frontend for the AI Music Generator application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Features

- Generate custom music from text prompts
- Use preset meditation sounds
- Adjustable duration (1-300 seconds)
- Download generated music as MP3
- Audio player for preview

## Build

To build for production:
```bash
npm run build
```
